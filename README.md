# Projeto SEGUP - Frontend

Estamos usando Typescript com Next e Tailwind.

## Pré-requisitos

Certifique-se de ter instalado o seguinte para não começar as frases "Aqui não tá rodando" e "Mas na minha máquina funcionou":

- **Node.js 24**: recomenda-se que não uses docker para o front se não estiveres acostumado a usá-lo, por causa de algumas configurações que devem ser feitas para que ele funcione como nativo.
- **Git**

## Como Rodar

Execute o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Abra [http://localhost:3000](http://localhost:3000) com um navegador.

E o Next.js tem Hot Reload, então ele detecta mudanças nos arquivos automaticamente e atualiza o navegador sozinho.
